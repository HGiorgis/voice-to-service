API docs

Use X-API-Key or Authorization: Bearer <key>. Base: /api/v1/

Process audio
POST /api/v1/process-audio/
Headers: X-API-Key: your-secret-key
Body: multipart/form-data — field audio (wav/mp3)

Response: amharic_text, english_text, category, confidence, timestamp.

Quickstart example

Python with requests:

import requests

url = "https://your-domain.com/api/v1/process-audio/"
headers = {"X-API-Key": "your-secret-key"}
files = {"audio": open("clip.wav", "rb")}
r = requests.post(url, headers=headers, files=files)
print(r.json())
